# Configuracion de Play Console

1. Crear la aplicacion con package `com.ruben.controlfinanciero` y categoria Finanzas.
2. Aceptar terminos y completar verificacion de la cuenta de desarrollador.
3. Configurar Play App Signing importando la app signing key actual; comprobar su SHA-256.
4. Crear y registrar una upload key separada.
5. Crear una cuenta de servicio, habilitar Android Publisher API y conceder solo permisos de release.
6. Completar App access, Data Safety, Financial features, audiencia, anuncios, content rating y politica de privacidad.
7. Crear primero un track interno en estado draft.

No se ha realizado ninguno de estos cambios externos desde el repositorio.
