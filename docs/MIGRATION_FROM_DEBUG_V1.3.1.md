# Migracion desde v1.3.1 debug

v1.3.2 inicia la firma Android release permanente. Por este cambio de certificado, no puede instalarse encima de la APK debug v1.3.1.

1. Comprueba que tus datos esten sincronizados y crea un respaldo desde la app cuando este disponible.
2. Conserva tus credenciales y confirma que el correo de acceso este verificado.
3. Desinstala v1.3.1 debug.
4. Descarga `control-financiero-ruben-v1.3.2.apk` desde la Release oficial.
5. Verifica opcionalmente el SHA-256 publicado e instala la APK.
6. Inicia sesion y revisa cuentas, movimientos y saldos.

Las versiones posteriores podran actualizarse sobre v1.3.2 porque reutilizaran esta misma firma release.
