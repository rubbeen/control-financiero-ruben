# Pruebas

Pruebas manuales obligatorias:

- Iniciar backend.
- Iniciar frontend.
- Confirmar `data/finance.db`.
- Confirmar categorias iniciales.
- Crear, editar y eliminar movimiento.
- Revisar dashboard.
- Revisar analisis mensual.
- Revisar comparativas.
- Generar PDF.
- Exportar JSON y CSV.
- Importar JSON sin duplicados.
- Cambiar URL del backend en Ajustes.
- Ejecutar configuracion Android con Capacitor.
