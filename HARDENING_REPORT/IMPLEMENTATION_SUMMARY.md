# Resumen de implementacion

v1.3.0 retira FastAPI, aisla Firestore por UID, agrega reglas probadas y migracion dry-run, reemplaza IDs por contadores transaccionales, corrige saldo/ajustes/transferencias, incorpora cuenta activa global, consultas acotadas, paginacion, React Query, analitica pura, presupuestos por categoria, rutas y Atrás, respaldo cifrado, PDF diferido, actualizacion verificada y Android endurecido.

La UI conserva cinco destinos principales, skeletons estables, estados de sincronizacion, objetivos tactiles de 44 px y COP. El asesor usa reglas y calculos; no afirma IA generativa.
