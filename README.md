# Control Financiero Ruben

Aplicacion de control financiero personal en COP para Android y web.

Incluye:
- Firebase Firestore para datos en la nube.
- Cuentas separadas para manejar balances independientes.
- Ingresos, gastos, presupuestos, categorias e historial.
- Analisis financiero y recomendaciones tipo IA dentro de la app.
- Actualizaciones por GitHub Releases.

APK inicial publicado en la seccion Releases de este repositorio.
